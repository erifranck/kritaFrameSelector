Frame Selector

A Krita plugin for selecting animation frames from the timeline and cloning them to other positions.

Features

  • Frame List: Browse all keyframes in the active layer with thumbnail previews.
  • Frame Preview: See a larger preview of any selected frame.
  • Clone Frames: Clone a frame's content to the current timeline position.
  • Multi-layer support: Works with the currently active animation layer.

Usage

  1. Open the Frame Selector docker from Settings > Dockers > Frame Selector.
  2. Select an animation layer in your document.
  3. The panel will show all keyframes with thumbnails.
  4. Click a frame to preview it.
  5. Navigate to the desired position in the timeline.
  6. Click "Clone Frame Here" to clone the selected frame to the current position.
